# LMS MERN - Learning Management System

This is the codebase for the production-ready Learning Management System (LMS) designed for schools, academies, coaching centers, and institutes.

## Project Structure

This project follows a monorepo setup structure:
- **`client/`**: React + Vite + Tailwind CSS frontend application.
- **`server/`**: Node.js + Express.js + Socket.io backend API.

## Features Included
- **Multi-Server Deployment Ready**: Dedicated databases per deployment.
- **Real-Time Communication**: Group chat, class broadcasts, and push notifications via Socket.io.
- **Analytics Dashboard**: Aggregated insights for Students, Faculty, Finance, and Attendance.
- **Academic Management**: Classes, sections, subjects, assignments, exams, and attendance.
- **Document Generation**: Automated PDF generation for ID cards, Certificates, Fee Vouchers, and Payment Receipts.
- **Virtual Learning**: Quizzes and recorded lectures.
